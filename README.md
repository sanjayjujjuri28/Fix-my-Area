# Fix My Area

Fix My Area is a MERN Stack web application that allows users to report and track local civic issues such as garbage problems, road damage, water leakage, and other public complaints.

## 🚀 Live Links

Frontend: https://your-vercel-link
Backend: https://your-render-link

## 🛠 Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* Mongoose
* REST API

## ✨ Features

* Submit complaints with details
* Upload images of issues
* Track complaint status
* Admin management system
* MongoDB database integration
* Responsive user interface

## 📌 Project Type

Hackathon Project – Built to solve real-world civic problems using technology.
