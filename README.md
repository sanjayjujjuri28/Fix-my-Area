# Fix My Area

Fix My Area is a MERN Stack web application that allows users to report and track local civic issues.

## 🌐 Live Demo

Frontend: https://fix-my-area-eta.vercel.app/

## 🛠 Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## ✨ Features

* Report local issues
* Upload images
* Track complaint status
* Admin management
* Real-time database using MongoDB

## 📌 Hackathon Project

Developed during a Hackathon to solve real-world civic problems.
